# How to Setup

Have Python Installed (I have 3.11)
Setup environment using `python -m venv .env`
Collect requirements using `pip install -r requirements.txt`
Use an extension Like [Quizlet Exporter](https://chromewebstore.google.com/detail/quizlet-exporter-export-f/lkoaedeomnobdibjfdfggjhoiiabpgkl) to gather a JSON of your quiz
Run using `python main.py your_quizlet.json`
