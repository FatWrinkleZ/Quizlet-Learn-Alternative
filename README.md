# How to Setup

<ul>
    <li>Have Python Installed (I have 3.11)</li>
    <li>Setup environment using <code>python -m venv .env</code></li>
    <li>Collect requirements using <code>pip install -r requirements.txt</code></li>
    <li>Use an extension like <a href="https://chromewebstore.google.com/detail/quizlet-exporter-export-f/lkoaedeomnobdibjfdfggjhoiiabpgkl" target="_top">Quizlet Exporter</a> to gather a JSON of your quiz</li>
    <li>Run using <code>python main.py your_quizlet.json</code></li>
</ul>
